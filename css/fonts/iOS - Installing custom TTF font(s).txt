# ------------------------------------------------------------
#
#   iOS - Installing custom TTF font(s)
#
# ------------------------------------------------------------


1) Download TTF font(s) to local iOS device's storage (from cloud storage, email, etc.)


2) Install as a profile onto iOS device using the "Fontcase" app @ https://apps.apple.com/us/app/id1205074470
