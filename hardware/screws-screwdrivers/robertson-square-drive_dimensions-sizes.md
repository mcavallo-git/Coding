# Screw/Screwdriver Dimensions, Sizes

  - ### Size Chart
    - | Size # |    Color |           Fits Screw Sizes | Tip Width (Metric) | Tip Width (SAE) |
      | -----: | -------: | -------------------------: | -----------------: | --------------: |
      |  `#00` | `Orange` |        `#1`, `#2`, `44928` |                    |                 |
      |   `#0` | `Yellow` |        `#3`, `#4`, `44989` |      ~ `1.59 mm +` |       `1/16" +` |
      |   `#1` |  `Green` |  `#5`, `#6`, `#7`, `45053` |      ~ `2.38 mm -` |       `3/32" -` |
