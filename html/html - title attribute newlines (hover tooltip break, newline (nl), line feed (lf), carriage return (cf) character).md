# Newline(s) in HTML hover tooltips (title attribute)

***

- Syntax:
  - `Line Feed` HTML Char:  ```&#010;```
  - `Carriage Return` HTML Char:  ```&#013;```

***

- Example:
  <div style="font-size:200%;" title="line 1&#010;&#010;line 3&#010;&#010;line 5">Hover over me</div>
