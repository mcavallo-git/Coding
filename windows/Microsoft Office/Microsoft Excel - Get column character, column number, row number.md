

## Microsoft Excel - Get target cell's row number(s) (1,2,3,...)
- ```=ROW()```


## Microsoft Excel - Get target cell's column number(s) (1,2,3,...)
- ```=COLUMN()```


## Microsoft Excel - Get target cell's column character(s) (A,B,C,...)
- ```=CHAR(64+COLUMN())```


<!-- ------------------------------ -->

## Column & Row (letter):
- ```=CONCAT(CHAR(64+COLUMN()),ROW())```
