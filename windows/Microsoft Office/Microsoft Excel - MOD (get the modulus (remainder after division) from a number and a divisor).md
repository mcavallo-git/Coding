# Microsoft Excel - Get the remainder after number is divided by divisor

```excel
=MOD(ROW(),32)
```

***
# Microsoft Excel - Base a set of rows off the last "checkpoint" row so many rows

```excel
=(ROW()-MOD(ROW(),32))
```
